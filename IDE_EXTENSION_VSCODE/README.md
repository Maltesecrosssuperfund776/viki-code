# VIKI VS Code Extension

This extension talks to the local VIKI CLI and API to submit tasks, inspect sessions, preview diffs, manage approvals, search the repo, look up symbols, and inspect logs.

## Usage

1. Start the local API with `viki up .`
2. Open the Command Palette and run `VIKI: Submit Task`
3. Use the VIKI activity bar panel for sessions and approvals
