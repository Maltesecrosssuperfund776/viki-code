# Code of Conduct

## Our Standard

We want VIKI Code to be a project where contributors can collaborate with technical honesty, respect, and professionalism.

Examples of expected behavior:

- give direct feedback without hostility
- focus on evidence and engineering substance
- welcome questions and reasonable disagreement
- respect review boundaries, security practices, and approval processes

Examples of unacceptable behavior:

- harassment, intimidation, or personal attacks
- bad-faith arguments or repeated disruptive behavior
- publishing private information or secrets
- knowingly submitting deceptive proof or misleading claims

## Enforcement

Project maintainers may remove or reject contributions, comments, or participation that violate this standard.

## Scope

This Code of Conduct applies to repository discussions, issues, pull requests, and project communication channels tied to VIKI Code.
